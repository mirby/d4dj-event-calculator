How To:

  • Add cards to your team. Expand the team details to add extra training levels per card.
  • Use the club, event and param sections to set additional details, if desired.
  • Use the resulting team power to calculate score (requires miyubot)
  • Enter the score in scoring section to see EP and other resource totals.
  • Click Save Data button to save your current team (including ET), event, club, and param values
  • ET levels are saved per card, so if you want to add extra training levels, you have to add the card, change the ET level, and then save data.

What is being saved?

  • This site uses localstorage (see docs here for more details)
  • Data is stored as a small JSON string to make it easier to resume your session if you close the browser window.
  • Clicking on the Remove All Saved Data string will clear all the JSON data.
  • If you open the browser devtools (F12 on Chrome), you can see the exact strings being saved when clicking the save button in the console tab.
