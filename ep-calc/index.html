How to use this site:

From the Add Cards to Team tab, you can build your team by adding cards from a list of all cards available in the game (English version). After choosing a card from the list, the card's statistics will display in the menu below the drop down. From here, you can pick the party slot (main or support) and then click the Add To Team button to add that card to you team. By expanding the Show Filters section, you can filter the card list by Unit or Style, and you can also sort by Total Power or by a target Parameter.

After adding to your team, you can view the extended stats by expanding the Main Team and Support Team sections at the bottom. From here, you can also add to the extra training levels for each card. Adding any cards to your team, or increasing the extra training levels, will reflect immediately on the team power stats on the top of the page.

From the Club / Event tab, you can set the club items for your team. The display, dj booth and discs can be character or unit. The remaining slots can be unit or style. As noted on the club item section, every potential item in the game can be chosen, but some items may not yet exist in the game, or you do not yet own them. You should double check the items you own in game when setting them here. Any club changes will display the Club Power (and the current parameters) from the team power section.

For the Event section, you can choose the event from the drop down list. The details of the event will be displayed in the table below it. When selecting an event, any power changes from the event bonus will be displayed on the team power section. Note, however, that the event power and additionally, the Total (with event) numbers, ONLY matter for certain types of events (Bingo during Battle Lives, and Medley during Task Medley). You will also notice that the event bonus does not affect the parameters, even though it appears to in game. This is because the event bonus doesn't actually give you more parameters that give you more event points, so I decided not to display it here.

From the Param Upgrade tab, you can set the parameter upgrades you've given to any characters so far. Like it is in game, the upgrades can be applied to the 3 parameters, and goes from 0 to 5%. Any parameter upgrade changes can be verified if you have a corresponding character set to your main or support team in the expanded team view.

From the Scoring tab, you can estimate the Event Points (EP) earned based on your team's power, skills, etc. You can enter the values from the game, or if you've built your own team on the site, you can click the Import Team button to bring over all the relevant fields. (Note that currently, passive skills are not imported, you will need to set them yourself). The calculations currently only use very specific songs (Cat's Eye for Poker, Slots and Raid, Here's the Light for Bingo, and a Direct Drive x4 medley for Medley).

For the Skills field, you may need to change the values around to get an accurate calculation. The import button will always bring in your team's current skills, but for some event types, it only uses the DJ slot's skills from all the members. For example, let's say your team's skills are 20%, 30%, 40%, and 50%, but you're playing in a Poker event. If you want to calculate a scenario where all the other members in your lobby have a DJ with a 50% skill, then you would want to change the 20,30,40,50 that got imported, into 50,50,50,50 to simulate this. The 5th skill (encore skill) will always default to using the highest skill from this list of four.

If you wish to just calculate EP by the score earned instead, you can do so from the Calculate EP by Score section. This takes in a score that you can either estimate or calculate from other methods (i.e. Miyubot).

On the bottom of the screen, you will be able to save and load from 5 different profiles. This allows you to keep using the same team if you close the browser. When you click save, your currently built team, club, and event are all saved to that profile only. Any extra trainings you've done to cards and any param upgrades are saved to all profiles. All extra trainings and param upgrades done, once saved, will be reflected in the card list if you choose any of the sorts. If you want to add extra training levels to more cards, you can add the card to your team, change the extra training value, and then save. If you ever want to start completely over, you can click the Remove All Saved Data button, which will delete all your profiles, as well as resets all the extra training and param upgrades to 0.

This site uses localstorage (see docs here for more details). The data is stored as a small JSON string on your computer. If you open the browser's devtools console (F12 on Chrome), you can see the string that gets saved every time you click the save button. Note that the saved data is treated similar to cookies, meaning if you clear your browser data, your saved data will be lost as well. In the future, there will be an option to export the profile information to a physical file.
